(1) Run ./scmakefile to initialize the scanner 
(2) Execute ./scanner.c filename.c to scan a file
    This creates a file named filename.cscn
(3) For debugging information, add the -debug flag like this: ./scanner.c -debug filename.c
    This generates an output file with extra details
